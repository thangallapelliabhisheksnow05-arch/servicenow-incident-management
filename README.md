# Incident Management System with Workflow Automation (ServiceNow)

## Project Overview
This project demonstrates the implementation of an Incident Management System using ServiceNow ITSM modules.

## Features
- Incident lifecycle management from creation to resolution
- Workflow automation using Flow Designer
- SLA configuration and tracking
- Business Rules for automatic assignment
- Client Scripts and UI Policies for form validation
- Reporting and dashboard creation

## Tools & Technologies
- ServiceNow
- ITSM
- JavaScript
- Flow Designer

## Outcome
Improved incident handling efficiency and automated workflows for faster resolution.
